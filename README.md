# Компьютерная игра «Airhockey» (Аэрохоккей)

* Лукашова Дарина Михайловна

###  Описание проекта
Данный проект разработан в рамках прохождения учебной ознакомительной практики на кафедре №42 ГУАП. Игра представляет собой симулятор аэрохоккея с возможностью локальной игры и противостояния искусственному интеллекту.

###  Видеодемонстрация механики игры
[Смотреть короткое видео игрового процесса в Google Drive](https://drive.google.com/file/d/1OccZ7DcW6qw49QJLzo6n3CDwVpnhZLuu/view?usp=sharing)

###  Технологический стек
* **Игровой движок:** Godot Engine 4.x
* **Язык программирования:** GDScript
* **Система контроля версий:** Git / GitHub

###  Функциональные особенности
* Реализовано два полноценных игровых режима: Одиночная игра против компьютера (ИИ) и локальный мультиплеер для двух игроков (PvP).
* Интегрировано два уровня сложности для бота-оппонента ("Легко" и "Сложно").
* Разработан алгоритм динамического контекстного управления системным указателем (автоматическое скрытие мыши во время матча и отображение в окнах меню/паузы).
* Присутствует полное звуковое сопровождение (клики интерфейса, фиксация гола, звуки триггеров).

###  Руководство по запуску проекта
1. Склонируйте данный репозиторий или скачайте его в виде ZIP-архива.
2. Установите и запустите игровой движок **Godot Engine 4**.
3. В окне управления проектами нажмите кнопку **"Импорт"** и укажите путь к файлу `project.godot` из данного репозитория.
4. Откройте проект и запустите его нажатием клавиши **F5**.
